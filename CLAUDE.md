# guitar-chordex

기타 코드 웹 앱 (TypeScript + React).

## 하네스: 기능 개발

**목표:** 기능 요구를 설계→구현→검증 에이전트 팀으로 처리하여, 통합 정합성이 검증된 React/TS 기능 코드를 산출한다.

**트리거:** 기능 개발·구현·수정·재구현·QA 재검증 등 guitar-chordex 기능 작업 요청 시 `chordex-feature-dev` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
